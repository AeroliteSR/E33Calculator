# E33Calculator

A simple calculator that outputs stats to json format.  

## NOTE:  
### "Rewards" are drops inherent to the enemy, whereas "Archetype Drops" are shared across all members of the enemy's Archetype. Some bosses, such as Simon, drop loot via a script (?) and not the actual victory screen. The output will still contain the archetype's loot tables regardless of this fact for the simple reason that I'm too lazy to manually map exceptions or figure out how the script system works.
